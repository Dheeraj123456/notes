# LaTeX

LaTeX is a typesetting system suitable for producing scientific, mathematical, and physical documents. This course covers commonly used math notation and KaTeX-compatible constructs.

## Topics

1. **Getting Started** — introduction, examples, inline math
2. **Accents and Emphasis** — accents, text emphasis, delimiter sizing
3. **Greek and Hebrew Letters** — Greek letters, Hebrew letters, other symbols
4. **Annotation and Layout** — cancel, overbrace, underbrace, vertical layout, overlap
5. **Delimiters** — parentheses, brackets, braces, absolute values
6. **Variable Size Symbols** — sum, product, integral, union, intersection
7. **Functions** — standard function names, math operators
8. **Logic and Set Theory** — quantifiers, set operations, implications
