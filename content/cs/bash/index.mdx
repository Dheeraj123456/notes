# Bash Scripting

Bash is a **Unix shell** and command language for the GNU operating system. This course covers the essentials for writing shell scripts.

## Topics

1. **Getting Started** — scripts, variables, arguments, functions, conditionals
2. **Parameter Expansions** — substitution, slicing, default values, transforms
3. **Arrays** — defining, indexing, iteration, operations
4. **Dictionaries** — key-value stores
5. **Conditionals** — integers, strings, files, regex
6. **Loops** — for, while, until, ranges, break, continue
7. **Functions** — defining, returning, error handling
8. **Options** — errexit, pipefail, glob options
9. **History** — expanding and re-running commands
10. **Miscellaneous** — arithmetic, redirection, heredoc, strict mode
