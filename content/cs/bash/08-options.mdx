# Bash Options

## Set Options

```bash
# Avoid overlay files
# (echo "hi" > foo)
set -o noclobber

# Exit upon error — avoid cascading errors
set -o errexit

# Unveil hidden failures
set -o pipefail

# Expose unset variables
set -o nounset
```

## Glob Options

```bash
# Non-matching globs are removed ('*.foo' => '')
shopt -s nullglob

# Non-matching globs throw errors
shopt -s failglob

# Case insensitive globs
shopt -s nocaseglob

# Wildcards match dotfiles ("*.sh" => ".foo.sh")
shopt -s dotglob

# Allow ** for recursive matches ('lib/**/*.rb' => 'lib/a/b/c.rb')
shopt -s globstar
```

## Strict Mode

```bash
set -euo pipefail
IFS=$'\n\t'
```
