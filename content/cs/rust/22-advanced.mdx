# Advanced Topics: Unsafe, FFI, SIMD, Allocators, Const Generics

## Overview

Advanced Rust features enable systems programming: **unsafe** bypasses borrow checker, **FFI** calls C/C++ code, **SIMD** exploits hardware parallelism, **custom allocators** control memory, and **const generics** add compile-time parameters.

---

## 1. Unsafe Blocks

### Unsafe Capabilities

```rust
unsafe {
    // 1. Dereference raw pointers
    let ptr: *const i32 = &42;
    let value = *ptr;

    // 2. Call unsafe functions
    libc::printf(b"Hello\n");

    // 3. Mutate static variables
    static mut COUNTER: u32 = 0;
    COUNTER += 1;
}
```

## 2. Raw Pointers

```rust
let x = 42;
let ptr_imm: *const i32 = &x as *const i32;
let mut y = 10;
let ptr_mut: *mut i32 = &mut y as *mut i32;

unsafe {
    *ptr_mut = 20;
    let arr = [1, 2, 3, 4, 5];
    let elem0 = *arr.as_ptr();          // arr[0]
    let elem1 = *(arr.as_ptr().add(1));  // arr[1]
}
```

## 3. FFI: Foreign Function Interface

```rust
extern "C" {
    fn strlen(s: *const u8) -> usize;
}

fn get_length(s: &str) -> usize {
    unsafe { strlen(s.as_ptr()) }
}
```

### C String Handling

```rust
use std::ffi::{CStr, CString};

let c_str = CString::new("hello").unwrap();
let c_ptr: *const i8 = c_str.as_ptr();

unsafe {
    let c_str_ref = CStr::from_ptr(c_ptr);
    let rust_str = c_str_ref.to_str().unwrap();
}
```

## 4. SIMD Intrinsics

```rust
#[cfg(target_arch = "x86_64")]
use std::arch::x86_64::*;

fn vector_add(a: &[f32; 4], b: &[f32; 4]) -> [f32; 4] {
    unsafe {
        let av = _mm_loadu_ps(a.as_ptr());
        let bv = _mm_loadu_ps(b.as_ptr());
        let sum = _mm_add_ps(av, bv);
        let mut result = [0.0; 4];
        _mm_storeu_ps(result.as_mut_ptr(), sum);
        result
    }
}
```

```
Scalar: 1 value per operation
SSE:    4 values (4× faster)
AVX2:   8 values (8× faster)
```

## 5. Global Allocators

```rust
use std::alloc::{GlobalAlloc, Layout};

struct MyAllocator;

unsafe impl GlobalAlloc for MyAllocator {
    unsafe fn alloc(&self, layout: Layout) -> *mut u8 {
        libc::malloc(layout.size()) as *mut u8
    }
    unsafe fn dealloc(&self, ptr: *mut u8, _layout: Layout) {
        libc::free(ptr as *mut libc::c_void);
    }
}

#[global_allocator]
static GLOBAL: MyAllocator = MyAllocator;
```

## 6. Const Generics

```rust
fn array_sum<const N: usize>(arr: [i32; N]) -> i32 {
    arr.iter().sum()
}

let sum = array_sum([1, 2, 3, 4, 5]);  // N = 5
// Compile-time: separate optimized version for N=5
```

## 7. Inline Assembly

```rust
use std::arch::asm;

fn add_one(mut x: i32) -> i32 {
    unsafe {
        asm!("add {0}, 1", inout(reg) x);
    }
    x
}
// Generates: add eax, 1
```

## Summary

| Feature | Safety | Performance | Use Case |
|---------|--------|-------------|----------|
| **Unsafe** | Programmer-checked | Best | FFI, atomics, optimization |
| **FFI** | Interop-dependent | Native | C integration |
| **SIMD** | Compiler-checked | 4-8× faster | Vector operations |
| **Custom Allocator** | Allocator-dependent | Customizable | Specialized memory |
| **Const Generics** | Compile-time | Zero overhead | Array sizes, compile-time math |
