# Data Structures

A **data structure** is a way of organizing and storing data for efficient access and modification.

## Topics

- [[cs/data-structures/arrays|Arrays]] — contiguous memory, index-based access
- [[cs/data-structures/linked-lists|Linked Lists]] — node-based, dynamic size
- [[cs/data-structures/trees|Trees]] — hierarchical structures

## Comparison

| Structure | Access | Search | Insert | Delete |
|-----------|--------|--------|--------|--------|
| Array | O(1) | O(n) | O(n) | O(n) |
| Linked List | O(n) | O(n) | O(1)* | O(1)* |
| BST | O(log n) | O(log n) | O(log n) | O(log n) |

*\*At head position*

## Why Data Structures Matter

Choosing the right data structure can dramatically impact algorithm performance. See [[cs/algorithms|Algorithms]] for more on how data structures enable efficient computation.
