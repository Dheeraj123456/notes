# Algorithms

An **algorithm** is a step-by-step procedure for solving a problem.

## Topics

- [[cs/algorithms/sorting|Sorting Algorithms]] — ordering data efficiently
- [[cs/algorithms/graph-algorithms|Graph Algorithms]] — traversal, shortest paths

## Why Algorithms Matter

Efficient algorithms can transform an impractical solution into a practical one:

$$
O(n!) \xrightarrow{\text{better algorithm}} O(n \log n)
$$

For $n = 100$, this is the difference between $10^{158}$ operations and ~460 operations.

## Prerequisites

Understanding [[cs/data-structures|data structures]] is essential for implementing most algorithms.
