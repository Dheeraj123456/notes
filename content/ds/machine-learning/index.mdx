# Machine Learning

**Machine learning** enables computers to learn patterns from data without being explicitly programmed.

## Topics

- [[ds/machine-learning/regression|Regression]] — predicting continuous values
- [[ds/machine-learning/neural-networks|Neural Networks]] — biologically inspired computation

## Types of Learning

1. **Supervised** — labeled data (regression, classification)
2. **Unsupervised** — unlabeled data (clustering, dimensionality reduction)
3. **Reinforcement** — reward-based learning
